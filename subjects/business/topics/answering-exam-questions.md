# Answering Exam Questions

## Exams Words

- Discuss
- Evaluate
- Role of
- Analyse
- Demonstrate
- Differentiate
- Identify
- Describe
- Explain

## Structure to Answer Questions

SEE Method:

1. State
2. Explain
3. Example

SEE Method Example:

**Manager:** <u>Responsible for day to day running</u> **<u>e.g. controls staff, roster, job list etc.</u>**

- Bold = State
- Underlined = Explain
- Bold and Underlined = Example
