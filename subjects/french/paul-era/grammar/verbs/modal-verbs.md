# The Modal Verbs

## To want to

Français,Anglais
vouloir,to want to
je veux,I want to
tu veux,you want to
il/elle/on veut,he/she/one wants to
nous voulons,we want to
vous voulez,you (all) want to
ils/elles veulent,they want to
